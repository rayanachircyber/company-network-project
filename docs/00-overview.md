# 00 – Project Overview

This project implements a scalable enterprise campus network in Cisco Packet Tracer.

Objectives:
- Apply Cisco hierarchical design (Access–Distribution–Core)
- Implement VLAN segmentation
- Enable Inter-VLAN routing
- Add redundancy using HSRP
- Use OSPF as dynamic routing protocol
- Provide Internet access using NAT
- Connect sites using a GRE tunnel
